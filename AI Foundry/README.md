# AI Foundry

Scripts and workflow definitions for Azure AI Foundry, including prompt flow and chatbot configuration.

## Contents

### Workflows

| File | Description |
|------|-------------|
| `Workflows/Chatbot-Model-Selection.yaml` | Prompt flow workflow definition for a chatbot that selects an AI model based on user input. |
| `Workflows/Chatbot-Model-Selection_v2.yaml` | Updated version of the chatbot model-selection prompt flow with refinements to the routing logic. |

## Usage

These YAML files are Azure AI Foundry prompt flow workflow definitions. Import them into an Azure AI Foundry project via the **Prompt Flow** section in [Azure AI Foundry Studio](https://ai.azure.com).
