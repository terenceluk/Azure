# AI Foundry Agent Instructions

System prompt and instruction files for Azure AI Foundry agents.

## Contents

| File | Description |
|------|-------------|
| `ARG-Query-Executor-Agent.txt` | System instructions for an AI Foundry agent that interprets natural language requests and executes Azure Resource Graph (ARG) queries to retrieve resource information across subscriptions. |

## Usage

These `.txt` files contain the system prompt / instructions to be pasted into an Azure AI Foundry agent's **Instructions** field when creating or editing an agent in [Azure AI Foundry Studio](https://ai.azure.com).
