# App Services

Scripts for managing Azure App Service configuration.

## Contents

| File | Description |
|------|-------------|
| `Add-Environment-Variables-App-Setting.ps1` | PowerShell script that adds or updates environment variables (Application Settings) on an Azure App Service, useful for bulk-configuring app settings from a list. |

## Prerequisites

- Azure PowerShell (`Az` module) with `Az.Websites`
- Contributor or higher permissions on the target App Service
